This project develops a Bayesian framework for estimating underground pipe
location and orientation from ground penetrating radar data, including
uncertainty quantification, prior and likelihood specification, and posterior
diagnostics.

Repository (private): [Yuxi0048/Uncertainty-Aware](https://github.com/Yuxi0048/Uncertainty-Aware)
